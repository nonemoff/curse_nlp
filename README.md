# NLP Курсовая работа

Автоматизированная система анализа текста информационного ресурса по предметной области "Алгоритмы консенсуса в распределённых системах".

## Структура проекта

```
curse_nlp/
├── corpus/              # PDF файлы для анализа
├── v2/                  # Основной код проекта
│   ├── main.py          # CLI интерфейс
│   ├── config.py        # Конфигурация
│   ├── requirements.txt # Зависимости
│   ├── core/            # Модули обработки
│   ├── data/            # Словари и данные
│   ├── cache/           # Кеш результатов (игнорируется git)
│   └── output/          # Результаты анализа
└── README.md            # Этот файл
```

## Быстрый старт

```bash
# Перейти в директорию проекта
cd v2

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или venv\Scripts\activate  # Windows

# Установить зависимости
pip install -r requirements.txt

# Скачать модели
python -m spacy download en_core_web_sm
python -c "import nltk; nltk.download('stopwords'); nltk.download('wordnet'); nltk.download('omw-1.4')"

# Запустить полный анализ
python main.py all
```

## Возможности

- **Извлечение текста из PDF** - автоматическое извлечение и определение языка
- **Частотный анализ** - закон Ципфа, характеристики M, N, K_R, K_I
- **Терминологический указатель** - TF-IDF, N-граммы, аббревиатуры
- **Именной указатель** - NER для русского и английского языков
- **Кеширование** - быстрая повторная обработка
- **Визуализация** - графики Ципфа и накопления лексики

## Команды CLI

```bash
python main.py extract    # Извлечение текста из PDF
python main.py analyze    # Частотный анализ
python main.py terms      # Терминологический указатель
python main.py names      # Именной указатель
python main.py all        # Полный пайплайн
python main.py status     # Статус кеша
python main.py clear      # Очистить кеш
```

## Результаты

Все результаты сохраняются в `v2/output/`:

- `extracted/` - извлечённые тексты из PDF
- `frequency_dict.csv` - частотный словарь
- `statistics.txt` - статистика анализа
- `term_index.csv` - терминологический указатель
- `abbreviations.csv` - аббревиатуры
- `name_index.csv` - именной указатель
- `graphs/` - визуализации

## Технологии

- **Python 3.13**
- **PyMuPDF** - извлечение из PDF
- **pymorphy3** - морфология русского языка
- **natasha** - NER для русского языка
- **spacy** - NER для английского языка
- **nltk** - NLP инструменты
- **pandas** - обработка данных
- **matplotlib** - визуализация
- **rich** - красивый вывод в терминале
